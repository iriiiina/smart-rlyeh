## General

React + Next.js application for monitoring home smart devices and sensors.

The project is under development and is not working in full capacity yet.

## Run the Application on http://localhost:3000

`npm run dev`

## Used NPM Modules

- Next.js + Sass: https://github.com/zeit/next-plugins/tree/master/packages/next-sass
- Reactstrap: https://reactstrap.github.io
- Font Awesome: https://github.com/FortAwesome/react-fontawesome
